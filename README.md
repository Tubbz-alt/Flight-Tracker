## Flight Tracker

This project allows you to record your flight in your own personal cloud storage.

### Features

- [X] Live status
- [X] Store flight paths
  - [X] JSON file
  - [X] SQLite
  - [ ] MongoDB
- [X] Store screenshots
  - [X] Azure
  - [ ] AWS
  - [ ] Imgur
- [X] Admin authentication
  - [X] Google account
  - [ ] Microsoft account
- [ ] Aircraft fleet
- [X] Image gallery
  - [X] Each flight
  - [ ] Each aircraft
- [X] Altitude chart
  - [X] Hover chart to see aircraft icon on map
