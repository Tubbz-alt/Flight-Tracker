## Flight Tracker

This project allows you to record your flight in your own personal cloud storage.

### Features

- [X] Live status
- [X] Store flight paths
  - [X] JSON file
  - [ ] MongoDB
- [X] Store screenshots
  - [X] Azure
  - [ ] AWS
  - [ ] Imgur
- [X] Admin authentication
  - [X] Google account
  - [ ] Microsoft account
- [ ] Aircraft fleet
- [ ] Image gallery
  - [ ] Each flight
  - [ ] Each aircraft
- [ ] Attitude chart
